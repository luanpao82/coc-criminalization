How to read the map

Each dot is a Continuum of Care at its state's centroid plus a stable deterministic jitter. Marker size encodes CoC size (log total beds); color encodes the selected variable. Color ranges are fixed across years, so the same color means the same value in FY22 and FY24.

Positions are for visual overview only — not a GIS-accurate CoC boundary map. Hover any dot for CoC identifier, leadership, and multi-variable context.
